\begin{abstract}

Presenting a computer programming problem or solution with a heigh-quality video is time consuming and it is not easy to update.

Technical content producers are strugle to find an efficient and interactive way to show their work.

This research introduces a tool, which with a presenter can create an interactive tutorial.

This tool shows programming code snippets in web based code editor and the content creatar can build a playable step by step "movie" with it.


\end{abstract}
